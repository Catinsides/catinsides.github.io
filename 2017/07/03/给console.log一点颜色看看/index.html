给console.log一点颜色看看

2017年7月3日

开发中少不了使用console.log进行调试，那么如何使打印出的内容具有颜色呢？

首先要说明的是，这里指的打印是在命令行中进行输出的，而不是在chrome控制台中。

答案是，把console.log()第一个参数设置为ANSI转义码即可, 第二个参数为需要打印的内容。
